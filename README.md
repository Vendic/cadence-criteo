# cadence/m2-criteo

### Summary
This extension provides Criteo OneTag Support For Magento 2.x

### Supported Events / Triggers
* ViewHome (cms index)
* ViewItem (product page)
* ViewList (category & search pages)
* ViewBasket (cart)
* TrackTransaction (purchase)
